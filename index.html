Git it!

Perhaps one of my favorite new things to do is use git. For my CS 3114 project I taught my partner git and we set up a private BitBucket repo for easy sharing. We heavily utilized it and had over 3 pages of commits/pushes/pulls along with quite a few 3am texts along the lines of "I FIXED THE BUG AHHHH PULL IN THE MORNING". There is nothing more comforting than knowing that if I mess up my code I can simply "git stash; git pull" and all will be well. I enjoy git so much that I took my website off of a traditional hosting service and am redoing it via github pages! The version control and flexibility is fantastic.
